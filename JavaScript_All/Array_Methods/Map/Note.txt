map() Method:
The map() method is an array method in JavaScript that is used to create a new array by calling a provided function on every element in the calling array.
It iterates over each item in an array, applies a transformation function to each item, and returns a new array with the results of applying the function to each item.

Map Object:
The Map object is a built-in JavaScript object introduced in ES6 that allows you to store key-value pairs and remember the original insertion order of the keys.
It provides methods for adding, removing, and retrieving elements by key.
It's useful for scenarios where you need to associate unique keys with specific values, and it offers efficient methods for managing key-value pairs.
Unlike arrays, which use integer indices to access elements, Map objects use any value (both primitive and object references) as keys.
Map objects are often used for storing data mappings, caching, or maintaining the order of elements in a collection.
