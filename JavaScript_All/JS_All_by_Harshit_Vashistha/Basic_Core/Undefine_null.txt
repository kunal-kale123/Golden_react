--------undefine----

Note :- undefine means variable decleare to kiya hai but usko koi value assign nahi ki hai vo sirf decleare hi hai 

Ex:1
const firstname;

O/P --->Error const ko ham open nahi rakh sakte usko kuch na kuch initially assign krna hi hota hai 

Ex:2

let firstname;

console.log(firstname) 

O/P -->undefine  --->let se jab variable decleare karte hai and usko koi value assign nahi karte hai tab vo undefine rhta hai 


Ex:3


let firstname;

firstname="Kunal"
